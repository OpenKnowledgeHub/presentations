Merge Konflikte

  • treten immer dann auf, wenn die gleiche Zeile in derselben Datei in unterschiedlichen Strängen bearbeitet werden
  • Konflikte müssen erst behoben werden, bevor ein Merge weitergeführt werden kann

Fortgeschrittene Funktionen

stash

  • mit stash können die offenen Änderungen des Arbeitsbereiches in einen Zwischenspeicher geschoben werden
  • der Stash ist wie ein Stapel aufgebaut, dessen Inhalte wieder angewandt werden können

Hands-On

revert

  • mit revert können die Änderungen eines Commits rückgängig gemacht werden
  • dabei wird die Git Historie nicht geändert

Hands-On

rebase

  • mit rebase werden die Commits eines Branches auf die eines anderen geschoben
  • dabei wird die Git Historie geändert

Hands-On

rebase --interactive

  • mit rebase --interactive wird ein Rebase auf denselben Branch ausgeführt
  • dadurch lassen sich Commits im Nachgang geändert werden
  • dabei wird die Git Historie geändert
